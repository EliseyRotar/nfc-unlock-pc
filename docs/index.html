📲 NFC Unlock PC

Tap your phone on a USB ACR122U reader and your Windows PC unlocks itself — no app, just a sticker.

MIT Licensed · Python · Windows 10/11 & Linux

What it does

Stick a cheap NFC tag (NTAG213/215) on the back of your phone. When you tap your phone on a connected ACR122U USB NFC reader, a small background service running on your PC recognises the tag's unique ID, retrieves your stored password from the OS keyring, and types it into the lock screen for you — instantly unlocking your session. Works on both Windows (via a SYSTEM scheduled task) and Linux (via a systemd --user service + xdotool/wtype).

How it works

┌──────────────┐ tap ┌────────────┐ PC/SC (pyscard) ┌──────────────────┐ │ Phone w/ NFC │ ──────────────▶│ ACR122U │ ───────────────────▶│ nfc-unlock-pc │ │ tag / sticker │ │ USB reader│ │ background service│ └──────────────┘ └────────────┘ └─────────┬─────────┘ │ UID matches enrolled tag? ▼ ┌─────────────────────────┐ │ SendInput → Winlogon │ │ (types password + Enter) │ └─────────────────────────┘

The ACR122U appears to Windows as a normal PC/SC smart-card reader, so the service talks to it with the pyscard Python library and reads each tag's UID via the standard FF CA 00 00 00 "Get UID" command. Because the Windows lock screen runs on an isolated secure desktop, the service runs as SYSTEM (via a Scheduled Task) so it can attach to that desktop and inject the keystrokes.

Get started

  1. Plug in your ACR122U and install the official ACS driver.
  2. Stick an NFC tag (NTAG213/215) on your phone.
  3. pip install -r requirements.txt
  4. python src\main.py enroll — tap your phone, enter your Windows password once.
  5. Run scripts\install_task.ps1 as Administrator to install the background service.
  6. Lock your PC, tap your phone — done.

Full setup instructions, troubleshooting and security notes are in the project README.

Project layout

reader.py

Talks to the ACR122U via PC/SC (pyscard), polls for tag UIDs.

config.py

Stores the enrolled tag UID and a DPAPI-encrypted Windows password.

enroll.py

One-time wizard: scan your tag, save your credentials securely.

unlocker.py

Attaches to the Winlogon secure desktop and types your credentials via SendInput.

service.py

The main loop — watches the reader and triggers the unlock.

install_task.ps1

Registers the service as a SYSTEM scheduled task that survives the lock screen.

Security notes

This is a convenience tool, not a high-assurance authentication system — anyone holding an enrolled (or cloned) tag can unlock the PC. Your password is encrypted with DPAPI and tied to this machine, but is still decryptable by SYSTEM/admin. If you want a polished, security-audited commercial alternative with native ACR122U + MIFARE support, see Rohos Logon Key.
